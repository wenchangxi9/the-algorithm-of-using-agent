# Evaluation pool

This folder contains the 258-note evaluation pool used by the experiments.

Files:

- `eval_claim_pool_258.csv`: note text, post text, and the official Community Notes status label used as the binary target.
- `ratings_cache_258.csv`: cached human ratings for reference statistics such as human helpful share.
- `fold_summary.csv`: original split metadata for the pool.

The original filename in the working directory contained an old project nickname. It has been renamed here to avoid mixing unrelated experiment names into this repository.
